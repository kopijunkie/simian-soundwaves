Style Sheet Changer

Style sheet changer toggle control via a link on the website's navigation bar which uses JavaScript to make the change. This feature also relies on creating a browser cookie to store the last selected style sheet to ensure that the style is persistent throughout the website as long as the visitor is on the website.

The technique and script was based largely on Alternative Style: Working With Alternate Style Sheets. But I decided to expand on it by making the style sheet changer into a single link toggle link. Based on the current active style sheet, this link's name would then change to the alternate style sheet's name (e.g. when the style 'Hello Sunshine' is clicked, sunny_style.css becomes the active style sheet and the link's (innerHTML) name changes to 'November Rain', which is rainy_style.css.

Some parts of the tutorial's script had to be greatly modified or taken out completely in order for it to work properly, particularly in order to get the chosen style to be persistent when changing between pages in the submission system. The cookie storing method turned out to be very important and very time consuming in this respect.

These are the code added in that make the name change and calls the function to switch style sheets:

function setActiveStyleSheet(title) {
document.getElementById('styler').href = title + '_style.css';
var i, a, main;
if (title == 'rainy') {
document.getElementById('style-changer').innerHTML = 'Hello Sunshine';
document.getElementById('style-changer').onclick = new Function("setActiveStyleSheet('sunny')");
}
else {
document.getElementById('style-changer').innerHTML = 'November Rain';
document.getElementById('style-changer').onclick = new Function("setActiveStyleSheet('rainy')");
}
createCookie("style", title, 365);
return title;
}

And this is the added code that makes the style sheet selection persistent throughout the website and also set the default style sheet if there is no cookie or style sheet already created/selected:

function styleSwitcher(title) {
var cookie = readCookie("style");
if (title == null && cookie == null) {
title = 'rainy'; /* Default Style */
}
if (title != null && cookie != title) {
cookie = title;
}
setActiveStyleSheet(cookie);
return false;
}

Automatic Image Slider (jQuery)

An automatic image slider made using the jQuery library was implemented in the Media page. The script and tutorial are from Automatic Image Slider w/ CSS & jQuery.

As I did not have enough time to learn jQuery in further enough detail to play around with the possible features, I only made very minor changes in the coding to get it working and displayed properly on the website:

  • hard coding the window width to match the set image reel width of 460px, as otherwise the image reel's display rotation calculation was totally off
  • removed the .paging { display: none; } part from the CSS and $(".paging").show(); as shown in the tutorial as the function to show the paging did not seem to work and I personally do not think the extra CSS/jQuery codes are necessary.

The image slider has been set to pause the image rotation "on hover" and the visitor can jump between the 4 photos by clicking on the corresponding paging number. The image slider is actually not limited to just 4 images, as the jQuery script actually calculates the length of the image reel based on number of images listed in the image_reel section of the HTML file.
